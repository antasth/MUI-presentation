The open-source React component library that implements Google's Material Design.

Introduction

  • What is material ui?

Features of React Material UI:

  • Highly customizable
  • Responsive
  • Fast loading
  • Themes
  • Backend friendly
  • Depreciation policy

How to install

Default installation

npm install @mui/material @emotion/react @emotion/styled

With styled-components

npm install @mui/material @mui/styled-engine-sc styled-components

Peer dependencies

"react": "^17.0.0 || ^18.0.0",

"react-dom": "^17.0.0 || ^18.0.0"

Fonts

Google Web Fonts

Material Icons font

Icons

npm install @mui/icons-material

Usage

Way 1:

import AccessAlarmIcon from '@mui/icons-material/AccessAlarm';

import ThreeDRotation from '@mui/icons-material/ThreeDRotation';

Way 2:

import { AccessAlarm, ThreeDRotation } from '@mui/icons-material';

Components API

The Component Api section describes all the examples, explanations and types of data passed to the MUI component, as well as CSS rules/classes for each component.

Button API

variant: 'contained' | 'outlined' | 'text' | string

size: 'small' | 'medium' | 'large' | string

color: 'inherit' | 'primary' | 'secondary' | 'success' | 'error' | 'info' | 'warning' | string

href: "https://google.com"

other props: "children, fullWidth, startIcon, endIcon, disabled"
